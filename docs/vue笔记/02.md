## 规范化
1. 检查组件名称是否符合要求(避免和HTML标签名冲突, 避免和内置组件冲突)
2. 规范化 props inject directive,将数组或者对象规范化为统一的结构
3. props 的初始化优先于 data 的初始化

```javascript
vm.$options = mergeOptions(
  // resolveConstructorOptions(vm.constructor)
  {
    components: {
      KeepAlive
      Transition,
      TransitionGroup
    },
    directives:{
      model,
      show
    },
    filters: Object.create(null),
    _base: Vue
  },
  // options || {}
  {
    el: '#app',
    data: {
      test: 1
    }
  },
  vm
)
```
